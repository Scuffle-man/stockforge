## 📌 文件标识说明

本仓库文件使用前缀图标/缩写标记来源与状态：

### 基础状态

| 图标   | 缩写   | 说明                                |
| ------ | ------ | ----------------------------------- |
| 🤖❓   | [AI?]  | AI 生成，未验证                     |
| 🤖✅   | [AI✓] | AI 生成，已验证，按步骤执行必定成功 |
| ✍️❌ | [ME✗] | 自己编写，不能正常工作              |
| ✍️✅ | [ME✓] | 自己编写，已验证可用                |

### 扩展状态（选用）

| 图标   | 缩写    | 说明                       |
| ------ | ------- | -------------------------- |
| 🤖⚠️ | [AI!]   | AI 生成，需修改/调整才能跑 |
| 🤖🔄   | [AI~]   | AI 生成，需要二次开发      |
| ✍️🚧 | [WIP]   | 施工中，功能未完成         |
| 🧪     | [EXP]   | 实验性代码，仅供参考       |
| 🗑️   | [DEPR]  | 已废弃，不再维护           |
| 📦❓   | [PORT?] | 移植代码，未验证           |

> 文件名示例：`🤖✅_deploy.sh` 或 `[ME✓]_parser.py`
